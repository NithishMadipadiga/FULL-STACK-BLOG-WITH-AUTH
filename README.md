# Full-Stack Blog Application with Authentication  A simple, secure, and clean full-stack Blog web application built using **Node.js, Express, MongoDB, and EJS**. This application features user registration, secure session-based password validation, and full CRUD capabilities for creating and reading blog articles.
## 🚀 Features

- **User Authentication:** Secure registration and login processes backed by `bcryptjs` encryption.
- **Session Management:** Persisted user states using `express-session` middleware to handle secure UI rendering dynamically.
- **Protected Routes:** Only registered users can access the write/publish section.
- **Relational Datamodels:** Leverages MongoDB object references to dynamically map authors to their respective blog posts.
- **Minimalist Clean UI:** Utilizes `water.css` for clean styling directly out-of-the-box without large asset overheads.

## 🛠️ Tech Stack

- **Backend:** Node.js, Express.js
- **Database:** MongoDB (via Mongoose ODM)
- **Frontend Engine:** EJS (Embedded JavaScript)
- **Security:** BcryptJS, Session-cookies
